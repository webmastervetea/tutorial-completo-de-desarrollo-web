Guía Maestra de Desarrollo Web Profesional

Optimización, SEO, Estilos y Animaciones

1. Imágenes de Próxima Generación

Utiliza formatos modernos para reducir el peso hasta un 50%:

  • WebP: 25-34% más ligero que JPEG.
  • AVIF: El estándar de oro actual.
<picture>
  <source srcset="imagen.avif" type="image/avif">
  <source srcset="imagen.webp" type="image/webp">
  <img src="imagen.jpg" loading="lazy" width="800" height="600" alt="...">
</picture>

Regla de oro: Usa loading="lazy" para imágenes fuera de pantalla, pero nunca para la imagen principal (LCP).

2. Semántica y SEO Técnico

Google lee tu web como un árbol de nodos. Usa etiquetas con significado:

  • <main>: Núcleo del contenido.
  • <article>: Contenido independiente.
  • JSON-LD: Formato recomendado para Datos Estructurados.

3. Fuentes Web y Unidades Relativas

Evita el uso de píxeles (px) y prefiere rem para accesibilidad.

Tip: Usa font-display: swap; para evitar que el texto sea invisible mientras carga la fuente (FOIT). 
h1 { font-size: clamp(2rem, 5vw + 1rem, 4rem); } /* Tipografía fluida */

4. Arquitectura CSS

Usa BEM (Block, Element, Modifier) para mantener la especificidad baja y el código reutilizable.

Concepto Ejemplo BEM Variables CSS
Bloque .card --card-bg
Elemento .card__title --text-color
Modificador .card--featured Dinámico con JS

5. Animaciones a 60 FPS

Para animaciones fluidas, anima solo transform y opacity. Esto utiliza la GPU y evita el reflow del navegador.

@keyframes flotar {
  0% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
  100% { transform: translateY(0); }
}
