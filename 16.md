## 1. El Poder de la Semántica HTML5

Google no "ve" tu web como un usuario; la "lee" como un árbol de nodos. Usar los elementos correctos ayuda a Google a identificar qué partes de tu código son ruido (menús, pies de página) y cuáles son el contenido principal.

### Estructura de Prioridad

Sustituye los clásicos `<div>` genéricos por etiquetas con significado:

* `<main>`: El núcleo del contenido de la página. Solo debe haber uno por documento.
* `<article>`: Para contenido que tiene sentido por sí mismo (posts, noticias).
* `<section>`: Agrupa contenido relacionado.
* `<aside>`: Para contenido tangencial (barras laterales, publicidad).
* `<h1>` a `<h6>`: La jerarquía lógica. **Solo debe haber un `<h1>` por página.**

## 2. Etiquetas Meta Esenciales (Head SEO)

Las etiquetas meta no solo sirven para el buscador, sino para controlar cómo se visualiza tu sitio en redes sociales y navegadores.

### Las "Imprescindibles"

```html
<head>
  <title>Optimización SEO para Desarrolladores | Guía Técnica</title>

  <meta name="description" content="Aprende a mejorar el posicionamiento de tu web mediante semántica avanzada, etiquetas meta y datos estructurados.">

  <meta name="robots" content="index, follow">

  <link rel="canonical" href="https://tusitio.com/seo-tecnico">
</head>

```

## 3. Rich Snippets con JSON-LD (Datos Estructurados)

Esta es la técnica más avanzada para destacar en Google. Los datos estructurados le dicen a Google exactamente qué está viendo: un producto, una receta, un evento o una organización.

**JSON-LD** es el formato recomendado por Google (insertado en un script):

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "SEO Técnico para Desarrolladores",
  "author": {
    "@type": "Person",
    "name": "Tu Nombre"
  },
  "datePublished": "2024-05-20",
  "image": "https://tusitio.com/imagen-seo.jpg"
}
</script>

```

*Tip: Puedes validar tu código en la [Prueba de resultados enriquecidos de Google](https://search.google.com/test/rich-results).*

## 4. Open Graph y Twitter Cards

Para que tu web se vea profesional cuando alguien la comparta en WhatsApp, LinkedIn o X (Twitter), necesitas estas etiquetas:

```html
<meta property="og:title" content="SEO Técnico Avanzado">
<meta property="og:description" content="Domina la semántica web.">
<meta property="og:image" content="https://tusitio.com/og-image.jpg">
<meta property="og:url" content="https://tusitio.com/seo">
<meta property="og:type" content="website">

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="SEO Técnico Avanzado">

```

## 5. Accesibilidad como Factor de Ranking

Google premia los sitios accesibles. Un sitio accesible suele estar bien estructurado.

1. **Atributo `alt` en imágenes:** No es solo para lectores de pantalla; ayuda a Google Images a entender el contenido.
2. **Atributos `aria-label`:** Úsalos en botones que no tengan texto (como iconos de redes sociales) para darles contexto semántico.
3. **Idioma:** Asegúrate de declarar el idioma en la raíz: `<html lang="es">`.

## Resumen de Implementación

| Elemento | Propósito | Impacto SEO |
| --- | --- | --- |
| **H1 - H6** | Jerarquía de información. | Alto |
| **Canonical** | Evitar contenido duplicado. | Medio |
| **JSON-LD** | Aparecer con estrellas o fotos en Google. | Alto (CTR) |
| **Meta Description** | Atraer clics desde el buscador. | Alto (CTR) |

### ¿Cuál es el siguiente paso?

Para que Google rastree tu sitio eficientemente, es vital tener un archivo **`sitemap.xml`** y un **`robots.txt`** bien configurados.
