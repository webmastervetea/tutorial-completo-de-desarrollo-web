Optimización de Imágenes y Variables CSS

1. Optimización de Imágenes de Próxima Generación

Los formatos modernos como WebP y AVIF ofrecen una compresión superior:

  • WebP: 25-34% más ligero que JPEG.
  • AVIF: Hasta 50% más de ahorro que JPEG.

Implementación con <picture>

<picture>
  <source srcset="imagen.avif" type="image/avif">
  <source srcset="imagen.webp" type="image/webp">
  <img src="imagen.jpg" alt="Descripción" width="800" height="600">
</picture>

Lazy Loading y CLS

Usa loading="lazy" para carga diferida, pero evita usarlo en la imagen principal (LCP). Además, define siempre width y height para prevenir el salto de diseño (CLS).

2. Variables en CSS (Custom Properties)

Las variables CSS permiten crear temas y mantener un código limpio y escalable.

La Regla del "Token"

Evita nombres descriptivos como --rojo; usa nombres basados en la función:

  • Bien: --color-brand-primary.
  • Mejor: --surface-main.

Comparativa de Variables

Característica Custom Properties (CSS) Variables SASS ($)
Tiempo de ejecución Dinámico (navegador) Estático (compilado)
