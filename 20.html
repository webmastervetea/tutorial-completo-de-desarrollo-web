Arquitectura CSS: Metodología BEM (Block, Element, Modifier)

1. La Estructura de BEM

El nombre de una clase BEM se divide en tres partes bien diferenciadas para evitar conflictos de especificidad:

  • Block (Bloque): El componente independiente y reutilizable (ej: card, menu, button).
  • Element (Elemento): Una parte interna del bloque que no tiene sentido por sí sola (ej: card__title). Se une con dos guiones bajos __.
  • Modifier (Modificador): Una variante o estado del bloque o elemento (ej: card--featured). Se une con dos guiones medios --.

2. Anatomía de una Clase BEM en HTML

Así se vería el marcado para una tarjeta de perfil siguiendo esta metodología:

<article class="card card--featured">
  <img src="user.jpg" class="card__avatar" alt="User">
  <h2 class="card__title">Juan Pérez</h2>
  <p class="card__description">Desarrollador Full Stack</p>
  <button class="card__button card__button--active">Seguir</button>
</article>

3. Reglas de Oro para un BEM Correcto

A. Evita la estructura "Abuelo-Hijo-Nieto"

BEM debe ser plano. Solo debe referenciar a su bloque raíz, sin importar la profundidad en el HTML.

  • Mal: .card__body__title
  • Bien: .card__title

B. No uses BEM para todo

Si un elemento es global, conviértelo en un bloque independiente (ej: .btn) para reutilizarlo fuera de su contenedor original.

C. Especificidad baja

Se utiliza casi siempre una sola clase para dar estilo, evitando el uso de !important o selectores anidados complejos.

4. BEM y Preprocesadores (SASS)

El símbolo & en SASS facilita enormemente la escritura de clases BEM manteniendo el bloque agrupado:

.card {
  background: white;
  &__title { font-size: 1.5rem; }
  &__button { 
    background: blue; 
    &--active { background: green; }
  }
  &--featured { border: 2px solid gold; }
}

5. Comparativa de Metodologías

Característica CSS Convencional CSS con BEM
Especificidad Suele ser alta Muy baja (una clase)
Reutilización Dificultada por el HTML Fácil (bloques módulos)
Legibilidad Confusa en proyectos grandes Auto-explicativa
