## 1. Declaración y Ámbito (Scope)

Para que una variable sea global, se declara en la pseudo-clase `:root`, que representa el elemento de más alto nivel en el documento (el HTML).

```css
:root {
  /* Definición: --nombre-variable: valor; */
  --primary-color: #3498db;
  --bg-color: #ffffff;
  --text-color: #2c3e50;
  --padding-standard: 1rem;
}

.button {
  /* Uso: var(--nombre-variable); */
  background-color: var(--primary-color);
  padding: var(--padding-standard);
  color: white;
}

```

## 2. Creación de Temas (Claro/Oscuro) sin Duplicar Código

La magia de las Custom Properties es que puedes redefinir su valor dentro de una clase o una media query, y todos los elementos que usen esa variable se actualizarán automáticamente.

### Paso 1: Definir la paleta base

```css
:root {
  --bg: #ffffff;
  --text: #1a1a1a;
  --accent: #7209b7;
}

/* Redefinimos los mismos nombres de variables para el tema oscuro */
[data-theme="dark"] {
  --bg: #121212;
  --text: #f8f9fa;
  --accent: #4cc9f0;
}

```

### Paso 2: Aplicar las variables

```css
body {
  background-color: var(--bg);
  color: var(--text);
  transition: background-color 0.3s ease; /* Transición suave al cambiar */
}

h1 {
  color: var(--accent);
}

```

## 3. Detección Automática con `prefers-color-scheme`

Puedes hacer que tu web respete la configuración del sistema operativo del usuario automáticamente:

```css
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #121212;
    --text: #f8f9fa;
  }
}

```

## 4. Mantenimiento y Código Limpio: La Regla del "Token"

Para mantener un proyecto escalable, evita usar nombres descriptivos del color (como `--rojo`). Usa **Tokens de Diseño** que describan la función:

* **Mal:** `--blue: #0000ff;` (¿Qué pasa si mañana el color de la marca es verde?)
* **Bien:** `--color-brand-primary: #0000ff;`
* **Mejor:** `--surface-main: var(--color-brand-primary);`

### Valores de Respaldo (Fallback)

Siempre define un valor de respaldo por si la variable no se carga correctamente:

```css
.card {
  /* Si --card-bg no existe, usará 'white' */
  background-color: var(--card-bg, white);
}

```

## 5. Manipulación con JavaScript

Cambiar de tema es tan sencillo como cambiar un atributo en el HTML. Esto permite crear el clásico interruptor (toggle) de modo oscuro:

```javascript
const toggleTheme = () => {
  const currentTheme = document.documentElement.getAttribute('data-theme');
  const targetTheme = currentTheme === 'dark' ? 'light' : 'dark';
  
  document.documentElement.setAttribute('data-theme', targetTheme);
};

```

## Comparativa: Variables CSS vs. Preprocesadores (SASS)

| Característica | Custom Properties (CSS) | Variables SASS ($) |
| --- | --- | --- |
| **Tiempo de ejecución** | Dinámico (en el navegador). | Estático (compilado a CSS). |
| **Media Queries** | Se pueden cambiar dentro de la query. | No pueden cambiar de valor en la query. |
| **JavaScript** | Accesibles y modificables. | Invisibles para JS. |
| **Soporte** | Todos los navegadores modernos. | Requiere compilación previa. |
